# 豆豆 - AI 自媒体工具

> Chrome 扩展，人人都能做自媒体

- [访问官网](https://www.undsky.com/doudou)

- [AI 画布](https://undsky.com/doudou_canvas)

豆豆是一款面向自媒体创作者的 Chrome 扩展工具，集成文章采集创作、图片/视频处理、AI 画布、极客工具等能力，帮助创作者从选题、采集、创作、处理素材到多平台分发，完成一站式内容生产流程，让任何人都能够在自己的赛道上持续创作！

![豆豆](assets/doudou_show.png)

## 核心能力

- **文章采集创作**：支持 Markdown 编辑、文章采集、AI 复刻、AI 改写、内容检测与 DOCX 导入导出。
- **全平台分发**：支持抖音、小红书、知乎、微信公众号、今日头条等 30+ 主流内容平台。
- **AI 画布**：通过可视化节点流编排 AI 能力，像搭积木一样完成图文、音视频和素材处理工作流。
- **图片/视频处理**：内置视频剪辑、视频拆解、视频合成、字幕合成、提取视频帧、转 GIF、分辨率调整、色彩调整、水印等工具。
- **极客工具**：提供页面截图、导出 Cookies、生成二维码、JSON 编辑器、CORS Unblock、网络抓包等实用工具。

## 安装

![安装](assets/install.png)

### 安装步骤

1. 下载或克隆本仓库到本地。
2. 打开 Chrome 浏览器，进入扩展程序管理页。
3. 开启「开发者模式」。
4. 点击「加载已解压的扩展程序」。
5. 选择豆豆项目目录完成安装。
6. 安装完成后，在浏览器扩展栏固定豆豆图标即可开始使用。

## 常见使用场景

- **对标文章创作**：采集参考文章，使用 AI 复刻和改写生成新内容，再同步发布到多个平台。
- **短视频生产**：用 AI 生成图片、视频和语音，组合字幕、音频、水印和分辨率处理后导出成片。
- **素材批处理**：提取视频帧、转 GIF、提取音频、裁剪片段、添加水印，快速完成素材加工。
- **提示词工作流**：通过反推提示词、AI 绘图、AI 视频等节点持续迭代内容效果。
- **开发辅助**：使用截图、抓包、JSON 编辑、Cookies 导出、二维码生成等工具处理日常开发任务。
