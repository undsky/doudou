爬虫规则使用说明

规则配置

选项 说明
爬取链接 目标页面 URL
爬取选择器 用于匹配目标元素的 CSS 选择器
链接选择器 用于匹配链接的 CSS 选择器
日期选择器 用于匹配日期的 CSS 选择器
内容选择器 用于匹配内容的 CSS 选择器

特殊设置说明

  • 仅配置内容选择器：当只填写「内容选择器」而不填「爬取选择器」时，将跳过列表采集阶段，直接采集当前页面的内容。
  • 日期选择器属性提取：日期选择器支持 selector$attr 格式，$ 后为要读取的 HTML 属性名。例如 time.date$datetime 表示选取 time.date 元素并读取其 datetime 属性值作为日期。
  • 内容选择器多选：内容选择器支持逗号分隔的多个选择器，采集时会依次提取各选择器匹配的内容并合并。例如 h1.title, div.content 会分别提取标题和正文。

爬取模式

模式 说明

滚动 自动滚动页面加载更多内容，可设置滚动次数和距离
分页 自动翻页爬取，可设置分页选择器和分页次数
